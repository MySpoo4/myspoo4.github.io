2. System‚Äôs Description & Core Features üíª

LockedIn evolved from a basic job board into a smart recruitment platform that personalizes job discovery, improves recruiter workflows, and enhances admin visibility.

For Job Seekers

Tailored Applications: One-click apply with a personalized note.

AI Recommendations: Suggested jobs based on skills & interests.

Interactive Map: Visualize job listings by location.

Commute Filtering: Filter by commute radius.

Privacy Controls: Manage profile visibility.

For Recruiters & Admins

Recruiter Dashboard: Kanban-style applicant management.

Messaging: Direct communication with candidates.

Saved Searches: Save candidate filters with notifications.

Admin Tools: Manage roles, users, and data exports.

CSV Export: Download user & job analytics.

Design Decisions

Django was selected for the backend for reliability and scalability, while Bootstrap and JavaScript enabled responsive and dynamic UI behavior (map interactivity, Kanban drag-and-drop, etc.).

Architecture & Diagrams

LockedIn follows an MVC-inspired Django structure. The frontend communicates with the backend via REST APIs to update data in real time.
