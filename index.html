2. System's Description & Core Features üíª

The web application is a professional job board designed to serve two primary actors: recruiters (posting jobs) and seekers (finding and applying to jobs). Sprint 1 focused on establishing core user roles and fundamental search capabilities.

Job Seeking and Filtering

Homepage Search: Users can search jobs by title, location, or type directly on the homepage.

Advanced Filtering: The "Find Jobs" tab offers extra criteria such as experience level and salary range.

Application Flow: Users can apply via a form when clicking a job listing and track their application history.

Recruiter and Admin Management

Job Posting: Recruiters can enter all necessary details for a new job.

Job Management: Recruiters can edit or delete their posted jobs.

Admin Oversight: Administrators can manage job roles, posts, and users via the dedicated Admin page.

Profile System

The profile page displays critical professional info such as role, skills, education, experience, links, and bio. Users can edit their profile, and recruiters can be contacted via email directly from their profile page.
